# Voting Application (Full Stack)

This is a **full stack voting application** where users can view polls, vote on them, and see results.  
The project is built using **Angular for the frontend** and **Spring Boot for the backend**, following a clean and structured architecture.

The main goal of this project is to demonstrate how a real world voting system works using modern web technologies.
